# 0.5.8 — 2026-06-02

## Changed

- **`@cyanheads/mcp-ts-core` ^0.9.16 → ^0.9.21** — framework adoption; see below for notable changes.
- **HTTP transport per-request log context** — per-request logs and traces now carry fresh request + trace/span IDs instead of the frozen boot context.
- **`fetchWithTimeout`** — strips query-string secrets (e.g. `?api_key=`) from error messages and logs.
- **`withRetry`** — fails fast on non-retryable errors; `ctx.fail` auto-populates the `retryable` flag.
- **Skill sync** — `add-tool`, `add-service`, `api-canvas`, `api-context`, `api-linter`, `api-utils`, `design-mcp-server`, `release-and-publish` updated; `api-mirror` and `orchestrations` added.
- **Scripts** — `devcheck.ts`, `check-skill-versions.ts`, `release-github.ts` added; `release:github` npm script added.
- **`vitest` ^4.1.7 → ^4.1.8**
